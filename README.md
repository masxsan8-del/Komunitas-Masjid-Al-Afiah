# Komunitas-Masjid-Al-Afiah
Website Komunitas Infaq &amp; Shodaqoh adalah platform digital yang memudahkan masyarakat untuk berbagi kebaikan melalui infaq dan shodaqoh secara aman, transparan, dan terpercaya. Setiap donasi ditampilkan secara realtime sebagai bentuk keterbukaan dan amanah dalam pengelolaan dana umat.
